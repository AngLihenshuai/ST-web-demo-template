How to Change Logo

Open the html files into a code editor. If you don't have any code editor, then you can download a free code editor. To download Notepad++ editor click here or Brackets editor click here. Replace "images/logo.png" with your logo image source url. Most of the updates are same for all pages. So we explained once here. Follow the screenshot

How to Change Text

Open the html files into a code editor and follow the steps. Replace the texts with your desired text. Most of the updates are same for all text and pages. So we explained once here. Follow the screenshot

How to Change Image

Images are located with <img src="image-source" alt=""> . Open html files into a code editor and follow the steps. Replace "images/watch-1.png" with your image source url. Most of the updates are same for all images and pages. So we explained once here. Follow the screenshot

How to Change Icon

Icons are located with <i class="icon-name"></i> . Open the html files into a code editor and follow the steps. We have used IcoFont icons click here for the icons list. Use Example: <i class="icofont-YourIconName"></i>. Most of the updates are same for all icons. So we explained once here. Follow the screenshot

How to Setup Contact Form

To receive emails from the contact form to your email's inbox folder, you have to need add your email address into contact.php file. Open contact.php file and replace <yourEmail@example.com> with your email address. Follow the screenshot

How to Setup Mailchimp Newsletter

To collect emails from newsletter to your mailchimp account, you have to need to add your mailchimp API key and Audience ID (Previous name was List).Open config.php file from newsletter folder, then add API key and Audience ID. Follow the screenshot
